# 起き上がり動作の評価と治療 〜螺旋・APA・最新エビデンスの統合〜

## 講義概要
- **講師**: 金子唯史（STROKE LAB）
- **対象**: 平均経験年数 7.5年（若手〜ベテラン）
- **時間配分**: 全90分
	- **第一部：座学セクション（30分）** 〜動作の解像度を上げる〜
	- **第二部：実技セクション（60分）** 〜スタッフ間実技動画で学ぶ極意〜

## スライド構成案

### 第一部：座学セクション（30分）
1. **1-3枚目**: タイトル・目標・アジェンダ
2. **4枚目**: イントロダクション：寝返りと起き上がりの「質」的な違い
3. **5枚目**: 運動学：なぜ「腹筋運動」は臨床で破綻するのか？
4. **6枚目**: 臨床病態：損傷部位と症状（Lesion-Symptom Mapping）
5. **7枚目**: 臨床分析：よく見る「起き上がり」の失敗4パターン
6. **8-10枚目**: メカニズム（螺旋・On Elbow・カウンターウェイト）

### 第二部：実技セクション（60分）
1. **11-12枚目**: 評価の視点（動作フロー・無視・Coreトーン）
2. **13-15枚目**: 介入戦略（螺旋誘導・On Elbow・下肢制御）
3. **16枚目**: 環境設定：エビデンスに基づく指導と翌日への繰り越し
4. **17-18枚目**: スタッフ実技：メカニズム再現・失敗体験・ハンドリング極意
5. **19枚目**: スキルアップ練習法：職場に持ち帰るために
6. **20枚目**: まとめ・質疑応答
