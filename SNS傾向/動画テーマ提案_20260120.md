# 今後の動画テーマ提案（2026/01/20）

## 1. 動画テーマ案とタイトル・構成

### 案①：【保存版・評価系】エビデンスに基づいた動作分析の決定版
Instagram分析の「評価・検査の徹底解説」に、最新の2025年エビデンスと筋電図データを加えた「信頼性No.1」を狙うコンテンツです。

*   **タイトル案：**
    *   【有料級】2025年最新エビデンスで解く「起き上がり」動作分析の極意
    *   なぜ腹筋を鍛えても起き上がれないのか？筋電図データが示す衝撃の事実
*   **ターゲット：** 実習生、新人〜5年目療法士
*   **構成案（15〜20分）：**
    1.  **イントロ：** 臨床でよく見る「失敗パターン（シーソー、カメ等）」の提示。
    2.  **理論（エビデンス）：** 「螺旋」「APA」の概念を解説。2025年の最新知見（右脳損傷の罠など）を紹介。
    3.  **可視化（sEMG）：** 筋電図データを用い、Co-contraction（同時収縮）によるエネルギーロスを視覚的に証明。
    4.  **評価の実技：** ベテランがどこを触り、何を見ているのか（Stiffnessの変化など）を言語化して解説。

### 案②：【権威性・実技系】30分ノーカットの臨床リアル
「30分治療シリーズ」の強みを活かし、講義案にある「螺旋ハンドリング」を実際の患者様（またはモデル）に適用する過程をすべて見せます。

*   **タイトル案：**
    *   【30分ノーカット】右脳損傷・Pusher傾向の「起き上がり」を劇的に変える螺旋ハンドリング
    *   重力を味方にする。脳卒中片麻痺への「起き上がり」介入戦略の全工程
*   **ターゲット：** 中堅療法士、技術向上を目指す若手
*   **構成案（30分〜）：**
    1.  **Before：** 努力的で非効率な起き上がり動作の提示。
    2.  **臨床推論：** 金子氏の視点を網羅した病態解釈の解説。
    3.  **介入（ハンドリング）：** On Elbowの再構築、前脛骨筋の活動遅延へのアプローチなど、マルチアングルで「圧のベクトル」を詳細に見せる。
    4.  **After：** 介入後の変化と、翌日へ繰り越すための環境設定の指導。

### 案③：【疾患特化・ブルーオーシャン】パーキンソン病（PD）攻略
PDの動画が伸びている分析結果と、講義案の「体軸性固縮」の知見を融合させ、競合の少ない領域を独占します。

*   **タイトル案：**
    *   パーキンソン病の「起き上がり」を自立させる！体軸性固縮を解く最新アプローチ
    *   【PDリハ】すくみ足を防ぐベッド高と、APA（予測的姿勢制御）を育てる自主トレ
*   **ターゲット：** 療法士、患者・家族層
*   **構成案（10〜15分）：**
    1.  **悩みへの共感：** PD特有の「体が丸太のように固まる」問題の提示。
    2.  **メカニズム：** 四肢の症状より「体軸性固縮」が自立を妨げる（2022年エビデンス）ことを解説。
    3.  **解決策：** 螺旋の動きを引き出すセルフエクササイズと、最新知見に基づく環境設定（ベッド高の最適解）。

## 2. 今後のコンテンツ制作における重要なポイント

1.  **「数値」と「波形」の活用：**
    研究項目にある「筋シナジー」や「CCI（同時収縮指数）」のグラフを動画内にインサートすることで、他のリハビリ動画にはない「科学的根拠」としての差別化が強調されます。
2.  **「失敗の言語化」を強みに：**
    講義案にある「あえて悪い例を演じ、努力しちゃう感触を解説する」手法は、視聴者の「なぜ上手くいかないのか？」という悩みにダイレクトに刺さります。
3.  **英語併記とSEO：**
    Instagram分析で「英語/日本語併記」がヒット要因とあったため、タイトルやサムネイルには「Spiral」「APA」「Core tone」などの用語を積極的に入れると、検索流入の強化が期待できます。
