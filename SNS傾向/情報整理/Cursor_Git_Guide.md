# 非エンジニアのための「Cursor × Git」活用ガイド

このガイドでは、日常業務を劇的に効率化するAIエディタ「Cursor」と、データの安全を守る「Git」について解説します。

---

## 1. Cursor（カーソル）：あなたの隣に座る「超優秀なAI秘書」

### 💡 例え：魔法の筆記用具
「こんな文章を書いて」「このデータを整理して」と伝えるだけで、あなたの代わりに下書きを作成し、間違いを見つけてくれるツールです。

| 項目 | メリット（いいところ） | デメリット（注意点） |
| :--- | :--- | :--- |
| **作業スピード** | **「〇〇して」と頼むだけ**で、数分かかる作業が数秒で終わります。 | AIがたまに**「もっともらしい嘘」**をつくことがあります。 |
| **知識の補完** | 専門知識がなくても、AIに相談しながら作業を進められます。 | 頼りすぎると、**自分で考える力が少し衰える**かもしれません。 |
| **ミスの発見** | 自分では気づかないような入力ミスをAIが即座に指摘します。 | 高性能なAIをフル活用するには**月額料金**がかかります。 |

---

## 2. Git（ギット）：最強の「共有ノート 兼 タイムマシン」

### 🛡️ 例え：絶対失敗しない保存システム
「誰が・いつ・どこを変えたか」をすべて記録します。間違えてもいつでも過去の状態に戻せるので、思い切った作業が可能になります。

| 項目 | メリット（いいところ） | デメリット（注意点） |
| :--- | :--- | :--- |
| **安心感** | 失敗しても**「昨日の状態」にボタン一つで戻せる**ので安心です。 | 最初に覚えるための**専門用語（コミットなど）**が少し必要です。 |
| **チーム連携** | 複数人で同じファイルを触っても、**上書きして消す心配がありません**。 | 二人が同じ場所を書き換えると**「どっちが正解？」の確認**が必要です。 |
| **保存性能** | クラウドに保存されるため、**パソコンが壊れてもデータは安全**です。 | 保存（コミット）作業を忘れると、そこまでの記録が残りません。 |

---

## 3. なぜセットで使うの？（相乗効果）

CursorとGitを一緒に使うことで、非エンジニアの方でも安心してクリエイティブな仕事ができます。

- **Cursorで挑戦！**：AIがサポートしてくれるから、難しいことにも取り組める。
- **Gitで守る！**：もし失敗しても、Gitで元通りにできるから怖くない。

---

## 4. 日常業務の4ステップ（これだけ覚えればOK）

1.  **Pull（プル）**：
    - 「まずは最新のノートを取り出す」
    - 他の人の最新の作業内容を自分のPCに反映させます。
2.  **Work（ワーク）**：
    - 「Cursor（AI）と一緒に作業する」
    - AIに相談しながら、資料作成やコード修正を行います。
3.  **Commit（コミット）**：
    - 「何をしたかメモを書いて保存する」
    - 「〇〇を修正」といった名前を付けて、作業の節目を記録します。
4.  **Push（プッシュ）**：
    - 「共有の棚に戻す」
    - 自分の作業内容をクラウド（GitHubなど）に送り、チーム全員に共有します。
