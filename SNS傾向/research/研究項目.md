# 筋電図を用いた生活期リハビリテーション効果の縦断的研究

## 1. 研究の目的
生活期（維持期）にある脳卒中、神経難病、脊髄損傷、整形疾患患者を対象に、sEMG（表面筋電図）解析を用いて運動制御戦略の変容を可視化し、リハビリテーション介入による機能回復プロセスの病態生理学的機序を解明する。

## 2. 対象疾患
- **脳卒中 (Stroke)**
- **神経難病 (PD: パーキンソン病 等)**
- **脊髄損傷 (SCI)**
- **整形疾患 (Knee OA, ACL損傷 等)**
※ 対象時期：生活期（急性期・回復期を経て在宅や施設で生活している段階）

## 3. 評価スケジュール（縦断的評価）
- **初回 (0ヶ月):** ベースライン測定
- **1ヶ月後:** 短期変化の確認
- **3ヶ月後:** 中期的な改善・適応の評価
- **6ヶ月後:** 長期的な効果の持続性および定着の評価

## 4. 臨床評価項目（Physical Performance）
- **歩行・バランス機能:**
    - TUG (Timed Up and Go Test)
    - 6m歩行テスト
    - 前方FRT (Functional Reach Test)
- **上肢機能:**
    - 上肢外転動作（自動可動域・スムーズさ）
    - 上肢挙上動作（自動可動域・スムーズさ）

## 5. 測定対象筋（sEMG）
- **下肢筋群:**
    - 前脛骨筋 (TA)
    - 腓腹筋内側頭 (MG)
- **上肢・体幹筋群:**
    - 三角筋前部繊維（上肢挙上動作の主動作筋）
    - 三角筋中部繊維（上肢外転動作の主動作筋）
    - 僧帽筋上部繊維（代償動作・安定化の指標）

## 6. 解析指標（EMG Metrics）
以下の指標を中心に、疾患ごとの特性（痙縮、固縮、AMI、代償動作）を分析する。

| 解析項目 | 主な分析対象・目的 |
| :--- | :--- |
| **Muscle Synergy** | 歩行時・挙上時の筋協調パターンの変容（モジュールの分離・結合） |
| **CCI / Co-contraction** | 関節の安定化戦略、およびPDの固縮やすくい足に伴う同時収縮の評価 |
| **Onset / Duration** | 動作開始の遅延（麻痺・AMI）や、活動遷延（弛緩不全・痙縮）の定量化 |
| **IEMG / Efficiency** | 努力度に対するパフォーマンス（歩行速度等）の比率。エネルギー効率の評価 |
| **Switching** | 歩行周期や反復動作における拮抗筋の切り替えのスムーズさ |
| **Asymmetry Index** | 左右の筋活動バランスの改善過程の評価 |
| **Asymmetry Index** | 左右の筋活動バランスの改善過程の評価 |
